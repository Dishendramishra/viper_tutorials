# Install Viper on Ubuntu

## Prerequisite
make sure you have python 3.6+ installed

## Download
clone or download this repo

give executable permissions to install_viper.sh

```
chmod +x install_viper.sh
```

then execute the shell script

```
./install_viper.sh
```
